# 📋 TODO: AI Review Bot — приложение без бэкенда (Android, Kotlin)

**Дата переработки:** март 2025  
**Цель:** Приложение, в котором **вся работа выполняется только в телефоне**. Своего бэкенда нет; приложение само обращается к Wildberries API и Yandex API, вся логика и данные — на устройстве.

---

## 📊 Общий обзор

- **Нет своего сервера** — не проектируем REST API для приложения, не деплоим бэкенд.
- **Внешние API с телефона:** Wildberries (отзывы + отправка ответов), Yandex (GPT, при необходимости Embeddings).
- **В приложении:** загрузка и хранение отзывов, классификация, RAG (локальный архив + поиск), вызов Yandex GPT, решение «авто/модерация», очередь модерации, статистика, настройки — всё в Kotlin и Room (или аналоге).

Задачи ниже — **анализ и подготовка** (без кода на этапе 0), затем пошаговая реализация.

## 📄 Связь с Python-проектом

Исходный AI Review Bot (Python) использовался как **референс логики** (классификация, RAG, решение, модерация). В Android-приложении эта логика **переносится в Kotlin и выполняется на устройстве**; Python-бэкенд для этого приложения **не используется**. Задачи по тестам, Docker, CI/CD для бэкенда к данному репозиторию не относятся — здесь только план и код Android-приложения без бэкенда.
