## Фаза 2: Данные и внешние API ✅ Выполнено

### Локальная БД (Room)

- [x] Сущности: отзыв (id, текст, рейтинг, артикул, статус, дата, ответ), очередь модерации (тот же статус ON_MODERATION)
- [x] DAO: вставка/обновление отзывов, выборка по статусу, пагинация, очередь модерации
- [x] Настройки (пороги, blacklist) — SecureSettings на EncryptedSharedPreferences

### Wildberries API

- [x] Retrofit: получение списка отзывов, отправка ответа на отзыв (WbFeedbackDto, WbAnswerRequest, WildberriesApi)
- [x] Репозиторий: вызов API → сохранение в Room; отправка ответа после одобрения; при синхронизации сохраняем локальные статусы

### Yandex API

- [x] Retrofit: вызов Yandex GPT (промпт + текст отзыва → ответ), YandexRepository
- [ ] (v2) Embeddings API при добавлении RAG

### Общее

- [x] Токены/ключи: хранить в SecureSettings (EncryptedSharedPreferences), не логировать в открытом виде
- [x] Обработка ошибок: Result<T>, таймауты в OkHttp, сообщения 401/429/5xx

**Результат:** Загрузка отзывов из WB, сохранение в Room, вызов Yandex GPT, отправка ответов в WB.

## Фаза 3: Логика на устройстве (domain) ✅ Выполнено

- [x] Классификация (правила на Kotlin): категория, сложность 1–5, blacklist из настроек (ReviewClassifier)
- [x] Движок решений: blacklist → MODERATE; иначе по порогам (сложность, уверенность, рейтинг) → AUTO_SEND или MODERATE (DecisionEngine)
- [x] Пайплайн: отзыв → классификация → Yandex GPT → решение → очередь модерации или отправка в WB (ReviewPipeline)
- [ ] (v2) RAG: эмбеддинги, поиск похожих, контекст для промпта

**Результат:** Полный цикл обработки отзыва на устройстве без бэкенда.

## Фаза 4: Настройки и экраны ✅ Выполнено

### Настройки

- [x] Ввод и сохранение: WB API-токен, Yandex API ключ, folder_id
- [x] Пороги: сложность, уверенность, минимальный рейтинг для автоответа
- [x] Редактируемый blacklist (список слов/фраз, по одному на строку)
- [x] Безопасное хранение секретов (SecureSettings / EncryptedSharedPreferences)

### Экраны

- [x] Список отзывов: данные из Room, фильтр по статусу, кнопка «Обновить» (синхронизация с WB)
- [x] Детали отзыва: текст, рейтинг, артикул, сгенерированный ответ; кнопки «Обработать», одобрить / отклонить
- [x] Очередь модерации: вкладка с фильтром «на модерации», те же действия (одобрить/отклонить)
- [x] Статистика: обработано, автоответы, в очереди, отклонено — подсчёт по Room
- [x] Навигация: нижняя панель (Отзывы | Модерация | Статистика | Настройки)
- [ ] (Опционально) Редактирование текста ответа перед отправкой; локальные уведомления

**Результат:** Пользователь настраивает токены, загружает отзывы, обрабатывает, модерирует и отправляет ответы в WB с телефона.
